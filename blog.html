Scope of var

JavaScript's `var` declares function-scoped variables, accessible within the entire function, but not block-scoped, potentially causing unexpected behavior. Favor `let` and `const` for block-level scoping.

Scope of let

In JavaScript, the `let` keyword declares variables with block-level scope, confining their accessibility to the nearest enclosing block or statement, enhancing code predictability and reducing variable conflicts.

Scope of const

In JavaScript, the `const` keyword declares variables with block-level scope like `let`, but it also ensures that the variable's value remains constant once assigned, preventing reassignment for immutability in the block.

What does it mean by Rest API

REST API (Representational State Transfer Application Programming Interface) is a set of architectural principles and constraints for designing networked applications. It is a widely used approach for building web services that allow different software systems to communicate with each other over the internet. Key principles of REST include:

1

Resources:

Everything is treated as a resource, and these resources are uniquely identified using URIs

2

Representation:

Resources can have multiple representations, such as JSON or XML, which clients can request based on their needs.

3

HTTP Methods:

It use standard HTTP methods like GET, POST, PUT, and DELETE to perform actions on resources.
